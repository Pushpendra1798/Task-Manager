# 🚀 Task Manager - DOM Explorer

A modern Task Manager application built using **HTML, CSS, and Vanilla JavaScript** to demonstrate core DOM concepts, Event Handling, Local Storage, and Browser Rendering concepts.

---

## 📌 Project Overview

This project was developed to explore and demonstrate important JavaScript DOM APIs and browser concepts through a practical Task Management application.

Users can:

* Create tasks
* Edit tasks
* Mark tasks as completed
* Delete tasks
* Persist data using Local Storage
* Toggle between Light and Dark themes
* Understand Event Propagation (Capturing & Bubbling)
* Learn Browser Rendering Pipeline concepts

---

## ✨ Features

### 📝 Task Management

* Add new tasks
* Edit existing tasks
* Delete tasks
* Mark tasks as completed
* Undo completed tasks

### 📊 Task Statistics

* Pending Tasks Counter
* Completed Tasks Counter

### 💾 Local Storage Support

Tasks are automatically saved in the browser's Local Storage and remain available after page refresh.

### 🌗 Theme Toggle

Switch between:

* Light Theme
* Dark Theme

### 🎯 Event Delegation

Uses Event Delegation for handling actions on dynamically created task cards efficiently.

### 🔄 Event Propagation Demonstration

Demonstrates:

* Capturing Phase
* Target Phase
* Bubbling Phase

with console logs for better understanding.

### 🌐 Browser Rendering Pipeline Visualization

Illustrates:

HTML → DOM Tree

CSS → CSSOM Tree

DOM + CSSOM → Render Tree

## 🚀 How to Run

1. Clone or download the project.
2. Open the project folder in VS Code.
3. Install the Live Server extension.
4. Right-click on `index.html`.
5. Select **Open with Live Server**.

The application will run in your browser.
